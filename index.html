Glosario de Java (Java 21)




Declaraciones y Modificadores

•package: Define el espacio de nombres del archivo.

•import: Importa otras clases o librerías.

•class: Declara una clase.

•interface: Declara una interfaz (contrato de métodos).

•enum: Declara una enimeración (lista de constantes).

•record: Declara una clase inmutable compactada (Java 14+).

•extends: Establece herencia de clase.

•implements: Implementa una o Varias interfaces.

•public: Acceso total desde cualquier lugar.

•private: Acceso restringido a la propia clase.

•protected: Acceso a la misma clase, paquete y subclases.

•static: Miembro que pertenece a la clase y no al objeto.

•final: Valor constante, método no sobrescribible o clase no heredable.

•abstract: Clase o método que debe ser completado por una subclase.

•native: Indica que el método está implementado en código nativo (C/C++).

•strictfp: Restringe los cálculos de punto flotante para asegurar portabilidad.



Tipos de Datos Primitivos

•byte: Entero de 8 bits.

•short: Entero de 16 bits.

•int: Entero de 64 bits (el estándar).

•long: Entero de 64 bits (usar sufijo L).

•float: Decimal de 32 bits (usar sufijo f).

•double: Decimal de 64 bits (el estándar).

•boolean: Valor lógico (true o false).

•char: Un solo carácter Unicode (16 bits).

•void: Indica que un método no devuelve nada.



Control de Flujo

•if / else: Bifurcación condicional.

•switch: Selección múltiple.

•case/default: Opciones dentro de un switch.

•yield: Devuelve un valor desde una expresión switch (Java 13+).

•while: Bucle que evalúa al final.

•do: Inicia un bucle con contador o iterador.

•for: Bucle con contador o iterador.

•break: Sale de un bucle o switch.

•continue: Salta a la siguiente iteración del bucle.

•return: Devuelve un valor y termina un método.



Manejo de Objetos y Memoria

•new: Crea una instancia de un objeto.

•this: Referencia a la instancia actual.

•super: Referencia a los miembros de la clase padre.

•instanceof: Comprueba si un objeto es de tipo determinad.

•var: Interferencia de tipo local (Java 10+).

•null: Representa una referencia vacia.



Manejo de Excepciones

•try: Bloque de código vigilando para errores.

•catch: Bloque que maneja la excepción.

•finally: Bloque que se ejecuta siempre al final.

•throw: Lanza una excepción manualmente.

•throws: Declara que un método puede lanzar excepciones.

•assert: Realiza una comprobación de depuración.



Concurrencia y Sincronización

•synchronized: Bloquea un método o bloque para un solo hilo.

•volatile: Garantiza que los hilos lean el valor más reciente de la memoria.

•transient: Indica que un campo no debe ser serializado.



Palabras Clave Especiales (Java 17-21)

•sealed: Clase cuya herencia está restringida.

•non-sealed: Abre una clase que era parte de una jerarquía sellada.

•permits: Lista las clases autorizadas para heredar de una sealed class.

•when: Cláusula de guarda en los nuevos switch (Java 21).



Métodos Esenciales de la Clase Object

•equals(): Compara si dos objetos son lógicamente iguales.

•hashCode(): Devuelve un código numérico único para el objeto.

•toString(): Representación en texto del objeto.

•getClass(): Obtiene la información de la clase en tiempo de ejecución.

•clone(): Crea una copia del objeto.

•wait() / notify() / notifyAll(): Control de hilos sobre un objeto.
