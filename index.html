Glosario de Java (Java 21)




Declaraciones y Modificadores

•package: Define el espacio de nombres del archivo.

•import: Importa otras clases o librerías.

•class: Declara una clase.

•interface: Declara una interfaz (contrato de métodos).

•enum: Declara una enimeración (lista de constantes).

•record: Declara una clase inmutable compactada (Java 14+)

•extends: Establece herencia de clase.

•implements: Implementa una o Varias interfaces.

•public: Acceso total desde cualquier lugar.

•private: Acceso restringido a la propia clase.

•protected: Acceso a la misma clase, paquete y subclases.

•static: Miembro que pertenece a la clase y no al objeto.

•final: Valor constante, método no sobrescribible o clase no heredable.

•abstract: Clase o método que debe ser completado por una subclase.

•native: Indica que el método está implementado en código nativo (C/C++).

•strictfp: Restringe los cálculos de punto flotante para asegurar portabilidad.
